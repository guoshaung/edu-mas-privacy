# EduMAS 个性化教育隐私保护平台

![EduMAS Logo](./docs/assets/edumas-logo.svg)

这是一个面向个性化教育场景的多智能体系统项目，核心强调“多智能体协作 + 本地隐私保护”的联合设计。系统目标是围绕学生学习全过程构建诊断、规划、辅导、检验与后续心理支持的一体化平台。当前仓库已经整理为单一主入口形式，适合演示、联调和后续继续扩展。

## 项目简介

项目总体面向以下学习流程：

- 个性诊断
- 学习规划
- 自适应辅导
- 知识检验
- 心理咨询与支持（后续扩展）

当前版本仍然属于原型阶段，并不是所有模块都已经完整实现。现阶段已经具备统一登录、学生端功能总览、引导式诊断、本地画像持久化、学生端与教师端分离等能力；同时也预留了多智能体调度、隐私网关、知识检验与心理咨询板块的后续扩展空间。

## 使用技术

- `Python 3.12`：本地主启动入口与后端原型开发
- `http.server`：轻量本地服务启动
- `React`：统一登录界面与前端交互逻辑
- `HTML / CSS / JavaScript`：当前学生端与教师端页面实现
- `localStorage`：学生本地诊断结果与画像持久化存储
- `Pydantic`：结构化数据建模
- `FastAPI` 及相关后端依赖：为后续接口层扩展预留
- `NumPy`、`Pandas`、`scikit-learn`：学习分析与教育数据处理支持
- `NetworkX`：图结构建模实验支持
- `Opacus`：差分隐私实验支持

## 当前功能范围

已实现或已展示的原型能力：

- 教师端 / 学生端统一登录入口
- 学生端四大功能总览页
- 个性诊断引导式采集
- 本地学生画像存储与本地图谱展示
- 教师端无法直接读取学生本地画像
- 面向隐私保护的本地优先交互方式

正在扩展或后续计划加入：

- 个性化学习规划自动生成
- 自适应辅导流程
- 知识检验与反馈闭环
- 本地隐私网关与服务端多智能体协同
- 心理咨询板块与情绪支持能力

## 主入口

在项目根目录运行：

```bash
python app.py
```

启动后访问：

```text
http://127.0.0.1:8000/login.html
```

进入统一登录页后，可以选择教师端或学生端。

## 当前目录说明

- `app.py`：项目主启动入口
- `login.html`：统一访问界面
- `login_app.jsx`：React 登录页逻辑
- `frontend/`：学生端、教师端及相关页面
- `gateway/`：隐私网关与权限控制相关代码
- `protocols/`：协议与共享定义
- `docs/assets/`：项目资源与 Logo
- `docs/guides/`：使用说明、部署说明、接口说明
- `docs/reports/`：阶段总结、测试报告、项目归档文档
- `scripts/maintenance/`：校验脚本与维护脚本

## 学生端流程

- 统一登录
- 学生功能总览
- 个性诊断
- 学习规划
- 自适应辅导
- 知识检验
- 心理咨询与支持（规划中）

其中个性诊断页会在学生本地浏览器中持久化保存画像相关数据，教师端默认不可直接查看这份本地画像。
