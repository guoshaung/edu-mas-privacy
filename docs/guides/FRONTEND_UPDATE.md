## 🔧 **技术实现**

### OCR识别流程

```javascript
// 1. 读取图片文件
FileReader.readAsDataURL(file)

// 2. Tesseract.js OCR识别
Tesseract.recognize(imageData, 'chi_sim+eng', {
    logger: m => console.log(m.progress)
})

// 3. 获取识别结果
result.data.text

// 4. 发送给教师端
window.opener.postMessage(uploadData, '*')
```

### 数据传输格式

```javascript
{
    type: 'student-upload',
    studentName: '张同学',
    studentId: 'stu_001',
    imageData: 'data:image/jpeg;base64,...',
    ocrText: '识别出的文字内容',
    timestamp: '2026-03-13 23:30:00'
}
```
