### 系统拓扑

```
┌──────────────────────────────────────────────────────────┐
│                   学习者域 (高信任)                        │
│                     A1: 学生档案                           │
│              唯一接触原始明文隐私的节点                      │
└─────────────────────┬────────────────────────────────────┘
                      │ 跨域请求（唯一通道）
                      ↓
┌──────────────────────────────────────────────────────────┐
│                 隐私网关 (控制平面)                         │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐        │
│  │  SRPG    │→ │  RBAC    │→ │  G-safeguard     │        │
│  │ (LDP)    │  │ (动态)   │  │    (GNN)         │        │
│  └──────────┘  └──────────┘  └──────────────────┘        │
└─────────────────────┬────────────────────────────────────┘
                      │ 受保护数据
                      ↓
┌──────────────────────────────────────────────────────────┐
│                  教学域 (低信任)                           │
│   AG2(风格)  AG3(辅导)  AG4(内容)  AG5(评估)              │
│            独立智能体，零横向直连                           │
└──────────────────────────────────────────────────────────┘
```

## 🔐 核心创新点

### 1. 数据层：SRPG + LDP
- **隐空间重构**: 剥离身份标识
- **LDP噪声**: 满足ε-差分隐私
- **隐私预算**: Opacus RDP记账器
- **文件**: `gateway/privacy_engine.py` (5,558 bytes)

### 2. 架构层：动态RBAC
- **状态机驱动**: 业务状态切换自动吊销权限
- **临时授权**: TTL + 访问次数限制
- **数据降维**: 最小权限原则
- **文件**: `gateway/rbac_manager.py` (8,396 bytes)

### 3. 防御层：GNN拓扑防御
- **实时监控**: 智能体交互拓扑图
- **异常检测**: GAT图注意力网络
- **动态剪枝**: 自动隔离污染源
- **文件**: `gateway/gnn_guard.py` (14,154 bytes)

### 4. 零横向直连
- **强制路由**: 所有跨域通信必须经过网关
- **物理隔离**: Docker容器隔离
- **审计日志**: 所有操作可追溯

### 5. AG5独立评估
- **完全独立**: 不读取辅导对话
- **防自判幻觉**: 独立考官机制
- **公平评分**: 仅基于标准答案
